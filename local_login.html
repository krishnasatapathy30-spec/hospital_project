Quick Local Login

This small form posts to your local server at 127.0.0.1:5000.

Need to start the server first? Use these commands:

E:\Hospital_project\venv\Scripts\Activate.ps1
E:\Hospital_project\venv\Scripts\python.exe E:\Hospital_project\app.py

Tip: open 127.0.0.1:5000/login in your browser once the server is running.
